HOW THIS PAGE WAS BUILT
This website's home page comes from my Neocities page, which has now been abandoned, though I might occasionally push code to it whenever I remember it. That page was developed so I could get a taste of vanilla CSS, since I normally use Tailwind. I thought that might be fun, but it was just time consuming! So I went back to Tailwind. (For those unaware, Tailwind CSS is a tool that just adds a bunch of CSS classes that usually do one thing which helps you build things faster. Really neat, might make a page about it at some point) Anyways, that page has 6 boxes on it, all for different websites/pages.

My main page isn't that interesting, so I'll skip onto this page. This “blog” was basically made because I was bored, and let's go over its parts. On the left of the page, you can see a really sweet navbar that you can hover over and see options appear.
The icons were sourced from Bootstrap and everything else was done by me. I'm very happy with it. For the bar itself, the CSS looks like this:
<
The nav properity tells screen readers when there's a list of links. This is helpful for accessibility, but functionally works the same as a div.
nav
Classes are used in websites to make things look nice. Tailwind provides many of these, and that's what's being used here.
class =
A collection of classes that are linked to CSS properties. Feel free to search some of these.
"fixed bg-zinc-700 w-16 h-full group hover:w-36 transition-all z-10" ><
End navbar.
/nav >
If you hover over the different parts, you can see what they do. It's set as a group so that I can trigger the icons to change to text by scaling down the icons to 0 and the text to 100, so it's all actually there, just layered on top of each other. The transition is set with a duration of 300ms for the expansion of the panel, which is Tailwind's default, and I thought that was fine. The text is set to turn black when hovered, and all of its code is here: 
            

<ul class="text-center">
<li class="scale-0 group-hover:scale-100 pb-4 hover:text-black transition-all duration-500 pt-2 cursor-default">
<a href="./blog">Blog</a>
</li>
<li class="scale-0 group-hover:scale-100 pb-4 hover:text-black transition-all duration-500 cursor-default">
<a href="./blog"Programming</a>
</li>
<li class="scale-0 group-hover:scale-100 pb-4 hover:text-black transition-all duration-500 cursor-default">
<a href="./blog"Other</a>
</li>
<li class="scale-0 group-hover:scale-100 hover:text-black transition-all duration-500 translate-y-[70vh] hover">
<a href="/">Home</a>
</li>
</ul>
The important bits are that when hovered (hover:) it sets the text to black. This is done with a duration of 500ms. Now the side bar is done, let's move onto the content of the page.
There's not too much here, so I'll go over it really quickly. The text is set to take of 50% of the screen, as is the image, and we have the date of creation stuck on the top right. The title's font is Ubuntu, and the body text's is Inter. What's more interesting are these little code boxes I've stuck throughout the page, like this:

These are done by creating a <div> and then setting it to have a different color compared to the background. I'll pad this a little bit, and then put the text in. colors: {

    'arg': '#9CDCFE',
    'string': '#CE9178',
    'greaterless': '#808080',
    'brackets': '#CE9178',
}
