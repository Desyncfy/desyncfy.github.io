Crazy Cattle 3D (Desync's Version)

 I did not make this game. This is a modified version of Crazy Cattle 3D designed to implement some extra features. I HIGHLY recommend you play the original first. If you enjoy that game, maybe consider leaving the original developer a tip. 

What is this?

CrazyCattle3D is one of the best games ever made. Go play it. This is a version of CrazyCattle3D with some extra features I wanted in the game.

What kind of features?

Here's a list of the currently implemented features:
  • Level unlock button
  • Warp to Environment.tscn (debug level)
  • Extra settings
And here are some features I'm planning to implement:
  • Speedrun timer/gamemode
  • Multiplayer (high priority)
  • More levels
